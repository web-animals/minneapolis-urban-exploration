## üìù Content Guidelines

### Clues
- **Solvable but not obvious** - Should take 5-15 minutes
- **Historically accurate** - Cite your sources
- **Safe locations** - No trespassing, no danger
- **Inclusive** - Accessible to people with different mobility levels

### Locations
- **GPS coordinates** (decimal degrees)
- **Historical context** (2-3 paragraphs)
- **Current status** (public/private, hours if applicable)
- **Safety notes** (parking, lighting, accessibility)
- **Photo tips** (best angles, times of day)

### Photos
- Your own photos preferred
- Respect copyright if using others
- No photos of private residences without permission

## üîß Technical Setup

### Prerequisites
- Git
- GitHub account
- Text editor (VS Code recommended)

### Getting Started

1. **Fork the repository**
2. **Clone your fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/minneapolis-urban-exploration.git
   cd minneapolis-urban-exploration
   ```
3. **Create a branch**
   ```bash
   git checkout -b add-new-clue
   ```
4. **Make your changes**
5. **Commit**
   ```bash
   git add .
   git commit -m "Add clue #6: The Falls"
   ```
6. **Push and submit PR**
   ```bash
   git push origin add-new-clue
   ```
